##配置Deployment
同样在_config.yml文件中，找到Deployment，然后按照如下修改：

比如我的仓库的地址是git@github.com:MaLilong/MaLilong.github.io.git，所以配置如下

deploy:
  type: git
  repo: git@github.com:MaLilong/MaLilong.github.io.git
  branch: master

##写博客、发布文章
新建一篇博客，执行下面的命令：

hexo new post "article title"


这时候在我的 电脑的目录下 ~\hexo\source\ _posts 将会看到 article title.md 文件

用MarDown编辑器打开就可以编辑文章了。文章编辑好之后，运行生成、部署命令：

hexo g   // 生成
hexo d   // 部署

当然你也可以执行下面的命令，相当于上面两条命令的效果

hexo d -g #在部署前先生成


踩坑提醒
1）注意需要提前安装一个扩展：

npm install hexo-deployer-git --save #在部署前先生成

如果没有执行者行命令，将会提醒

deloyer not found:git
